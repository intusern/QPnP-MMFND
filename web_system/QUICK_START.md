# QMMFND Web System - 快速启动指南

## 🚀 三步快速启动

### 第一步：准备环境
✅ 确保已安装 Python 3.7+ 和 Git
✅ 系统网络连接正常

### 第二步：运行启动脚本

#### Windows 用户
```
双击 run.bat 文件
或在命令行执行：
cd web_system
run.bat
```

#### Linux/Mac 用户
```bash
cd web_system
chmod +x run.sh
./run.sh
```

### 第三步：打开应用

**方式A：直接打开HTML文件**
```
在文件浏览器中找到 web_system/frontend/index.html
双击打开或右键 → 用浏览器打开
```

**方式B：在浏览器中打开**
```
Chrome/Firefox 地址栏输入：
file:///path/to/web_system/frontend/index.html

或访问API：http://localhost:5000
```

## 🔧 常见问题处理

### ❌ 问题：启动后提示"无法连接后端"

**解决方案：**
1. 检查是否还有命令行窗口打开（Windows）或终端窗口打开（Linux/Mac）
2. 确认看到"Flask is running on http://0.0.0.0:5000"的提示
3. 如果没有提示，检查是否出现错误：
   - 端口5000被占用 → 修改app.py中的port为其他值
   - Python环境错误 → 重新运行脚本

### ❌ 问题：登录失败

**解决方案：**
1. 确认已使用演示账户（admin / 123456）
2. 检查是否有拼写错误
3. 等待数据库初始化完成（首次启动较慢）
4. 尝试刷新浏览器页面

### ❌ 问题：图表不显示

**解决方案：**
1. 打开浏览器开发者工具（F12）检查错误
2. 确认网络连接正常
3. 尝试清除浏览器缓存并刷新页面
4. 使用最新版本的Chrome或Firefox

### ❌ 问题：检测功能不工作

**解决方案：**
1. 确认已登录
2. 检查是否输入了新闻内容
3. 查看浏览器控制台错误信息
4. 重启后端服务

## 🔐 安全建议

1. **修改JWT密钥**（生产环境）
   编辑 `backend/app.py`：
   ```python
   app.config['JWT_SECRET_KEY'] = 'your-new-secret-key'
   ```

2. **使用强密码**
   首次登录后修改演示账户密码

3. **定期备份数据**
   定期备份 `qmmfnd.db` 文件

4. **HTTPS部署**
   生产环境应配置HTTPS证书
